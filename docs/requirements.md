# 《这块新田》互动个人网站需求文档

## 项目目标

实现一个单页沉浸式互动个人网站《这块新田》。网站不需要后端、不需要登录，所有内容写在本地配置文件中。体验重点是从门外进入温暖小屋，按顺序触碰房间物品，逐步探索个人介绍，最后打开礼盒看到完整介绍和欢迎联系文案。

## 技术栈

- React
- Vite
- TypeScript
- Tailwind CSS
- Motion
- Zustand
- Howler.js

## 体验流程

1. 开场为门外画面，周围偏暗，门缝透出暖光。
2. Hover 门把手显示提示：“这里好像有一扇门。要推开看看吗？”
3. 点击门把手后门打开，显示“正在进入「这块新田」……”，黑屏转场 1-2 秒。
4. 进入小屋后显示欢迎吊牌，提示先转动八音盒。
5. 小屋物品按顺序解锁：八音盒、书架、跑鞋、泳镜、电脑、照片墙、礼盒。
6. 已解锁物品 hover 显示气泡提示，点击后播放动画并打开内容弹窗。
7. 弹窗关闭后掉出纸条，提示下一个探索物品并解锁下一关。
8. 最后点击礼盒，显示完整自我介绍、探索进度 100%、欢迎联系文案。

## 视觉方向

整体为温暖、可爱、轻微游戏感的互动房间。采用针毡羊毛艺术风格、手工制作感、柔软蓬松质感、微缩童话场景、暖色调电影光影、柔和散射光、木质元素和浅景深氛围。首版使用 CSS/DOM 插画模拟毛毡与手绘质感，画面保持简洁，不使用大量涂鸦。

## 首版范围

- 桌面端优先，移动端保证可浏览和可点击。
- Howler.js 音频结构保留，但首版不强制播放真实音频。
- 不放真实联系方式，只写“欢迎联系我，一起做些有趣的东西”。
- 不使用单张 AI 背景图热区，物品用独立网页元素实现。
